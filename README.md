# BeTran
Befunge-93 interpreter in pure Fortran.

Sample prorgrams included.

Compile by:    
          [complier] stack.f90 interp.f90 betran.f90 -o bertran
          
Run:    
      ./betran [file].bf

debug option available via commandline argument debug
i.e ./betran [file].bf debug

Tested on:     
Gfortran  5.4.1+    
     Ifort 17.0.4+
